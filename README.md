# 🎓 AAT PPT Generator

A web-based tool that helps students **automatically generate AAT presentations (PPTs)** using **AI (Gemini API)**.  
Students can log in, enter their problem statement, and download a professionally structured PPT in seconds.

---

## 🚀 Features

- 🔐 Student login system (Session-based)
- 🧠 AI-powered content generation using **Google Gemini API**
- 📊 Automatic PPT creation using **PptxGenJS**
- 🎨 Dark / Neon UI (student-friendly)
- ✍️ Auto-filled student details in PPT
- 📄 Slides generated based on problem statement

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS  
- **Backend:** Node.js, Express.js  
- **AI:** Google Gemini API  
- **PPT Generation:** PptxGenJS  
- **Hosting:** Vercel  
- **Sessions:** express-session
